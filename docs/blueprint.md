# **App Name**: VeriPresence

## Core Features:

- Secure Login: Students log in using their official ID and password.
- Biometric Verification: Utilize on-device fingerprint scanner or face ID for secure authentication upon each app launch.
- QR Code Scanner: Scan a 5-second rotating QR code displayed on the teacher's web application for attendance capture.
- Data Storage: Store scanned QR code data locally until manually sent to the teacher's web app. This involves storing student live location, name, and registration number
- Data Transmission: Manually send stored attendance data (location, name, registration) to the teacher's web app upon user action.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a secure and academic feel.
- Background color: Light grey (#E0E0E0) to ensure readability and a professional appearance.
- Accent color: Soft lavender (#E6E6FA) for highlighting interactive elements.
- Font pairing: 'Inter' (sans-serif) for body text and 'Space Grotesk' (sans-serif) for headlines, creating a balance of clarity and modernity.
- Use minimalist icons for navigation and data representation.
- Clean, straightforward layout emphasizing ease of use for quick attendance capture.
- Subtle transition animations for QR code scanning confirmation and data sending feedback.
